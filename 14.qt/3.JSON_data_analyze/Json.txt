{
	"pro":"河南省",
	"proCode":410000,
	"city":"郑州市",
	"cityCode":410100,
	"region":"中原区",
	"regionCode":410102,
	"error":""
}

以上形式的数据被称为Json数据，是一种轻量级的交换数据

Json中的数据有严格的格式规定

凡是以"{"开头 以"}"结尾包含的数据,被看作是一个Json对象(Object)
一个Json对象中可以存储若干个键值对
键值对的格式为 "键名":值

Json键值对中键后面的值,可以是 Json对象\Json数组\String\int\double\.....

凡是以"["开头 以"]"结尾包含的数据,被看作使一个Json数组(Array)
数组中一个存放多个值
